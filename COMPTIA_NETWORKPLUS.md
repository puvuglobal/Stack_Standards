# CompTIA Network+ N10-009 Certification Study Guide

> Vendor-neutral networking certification - Foundation for network careers

## Exam Overview

| Field | Details |
|-------|---------|
| **Exam Code** | N10-009 |
| **Version** | 4.0 |
| **Questions** | 90 maximum |
| **Length** | 90 minutes |
| **Passing Score** | 720 (on a scale of 100-900) |
| **Languages** | English, Japanese |
| **Certification Renewal** | 3 years |

---

## Exam Objectives (Domains)

### Domain 1.0: Networking Fundamentals (24%)

#### 1.1 - Compare and contrast the Open Systems Interconnection (OSI) model layers and their functions
- Application layer (Layer 7)
- Presentation layer (Layer 6)
- Session layer (Layer 5)
- Transport layer (Layer 4)
- Network layer (Layer 3)
- Data link layer (Layer 2)
- Physical layer (Layer 1)

#### 1.2 - Deploy and configure Ethernet networks with switches and routers
- Switch configurations
- VLANs
- Trunking
- Router configurations
- Default gateways

#### 1.3 - Compare and contrast network topologies
- Star topology
- Bus topology
- Ring topology
- Mesh topology
- Hybrid topology

#### 1.4 - Compare and contrast network types and characteristics
- LAN (Local Area Network)
- WAN (Wide Area Network)
- PAN (Personal Area Network)
- MAN (Metropolitan Area Network)
- WLAN (Wireless LAN)

#### 1.5 - Explain common ports and protocols
- HTTP/HTTPS
- FTP/SFTP
- SSH
- DNS
- DHCP
- SMTP
- IMAP/POP3
- SNMP
- LDAP

#### 1.6 - Explain IP addressing and subnetting
- IPv4 addressing
- IPv6 addressing
- Subnet masks
- CIDR notation
- VLSM (Variable Length Subnet Masking)
- NAT (Network Address Translation)
- PAT (Port Address Translation)

---

### Domain 2.0: Network Implementations (19%)

#### 2.1 - Compare and contrast networking devices
- Routers
- Switches (Layer 2/3)
- Firewalls
- Access points
- Load balancers
- IDS/IPS

#### 2.2 - Compare and contrast network cable types and specifications
- Copper cables (Cat5e, Cat6, Cat6a, Cat7, Cat8)
- Fiber optic cables (Single-mode, Multi-mode)
- Cable standards (TIA/EIA-568)
- Ethernet standards (10BASE-T, 100BASE-TX, 1000BASE-T, 10GBASE-T)

#### 2.3 - Given a scenario, install and configure network devices
- Router setup
- Switch configuration
- Firewall rules
- Wireless access point placement

#### 2.4 - Compare and contrast wireless networking concepts
- 2.4 GHz vs 5 GHz vs 6 GHz
- SSID and BSSID
- Channels and frequencies
- Antenna types
- MIMO
- Wi-Fi standards (802.11a/b/g/n/ac/ax)

---

### Domain 3.0: Network Operations (16%)

#### 3.1 - Explain the purpose and characteristics of network documentation
- Network diagrams
- IP addressing schemes
- Change management logs
- Baselines

#### 3.2 - Given a scenario, use appropriate network monitoring tools
- Packet capture tools
- Protocol analyzers
- SNMP tools
- NetFlow
- SIEM

#### 3.3 - Given a scenario, configure and verify DNS settings
- A records
- AAAA records
- CNAME records
- MX records
- PTR records
- DNS servers (primary, secondary, caching)

#### 3.4 - Given a scenario, configure and verify DHCP settings
- IP pools/scopes
- Reservations
- Lease time
- Router as DHCP server
- DHCP relay/forwarding

---

### Domain 4.0: Network Security (19%)

#### 4.1 - Compare and contrast security concepts
- Authentication vs Authorization
- Encryption methods
- Hashing
- PKI concepts
- Certificates

#### 4.2 - Compare and contrast common network attacks
- DoS/DDoS attacks
- Man-in-the-middle
- ARP poisoning
- MAC flooding
- DNS poisoning
- VLAN hopping

#### 4.3 - Given a scenario, implement network hardening techniques
- Port security
- DHCP snooping
- ARP inspection
- Access control lists (ACLs)
- Firewall rules
- VPN implementation

#### 4.4 - Compare and contrast remote access methods
- VPN (SSL, IPSec)
- Remote desktop
- SSH
- VNC

---

### Domain 5.0: Network Troubleshooting (22%)

#### 5.1 - Explain the troubleshooting methodology
- Identify the problem
- Establish a theory
- Test the theory
- Establish a plan
- Implement the solution
- Verify the solution
- Document findings

#### 5.2 - Given a scenario, troubleshoot common cable connectivity issues
- Cable tester results
- Open/short circuits
- Cross-talk
- EMI interference
- Attenuation
- Bad connectors

#### 5.3 - Given a scenario, troubleshoot common network issues
- IP configuration issues
- DNS resolution problems
- Routing issues
- Switching issues
- Wireless connectivity issues

#### 5.4 - Given a scenario, troubleshoot common network services
- DHCP issues
- DNS issues
- NTP issues
- Authentication issues

- Recommended: A+ or 9+ months of networking experience
- Multiple choice and performance-based questions (PBQs)
